# AI Video Generator
## 📋 Overview
An automated pipeline to generate Hindi videos from YouTube transcripts.

## 🚀 Features
- Extract YouTube video transcripts
- Generate Hindi scripts from transcripts
- Convert text to speech (TTS)
- Combine audio and video
- Upload to YouTube automatically

## 📦 Installation

### Prerequisites
- Python 3.8+
- FFmpeg installed
- YouTube API credentials

### Install Dependencies
```bash
git clone https://github.com/wajishah12/ai-video-generator.git
cd ai-video-generator
pip install -r requirements.txt
```
